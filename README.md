This project focuses on predicting **customer churn in a telecom company** using machine learning on the **Telco Customer Churn dataset sourced from Kaggle**. The objective is to identify customers who are likely to leave the service based on factors such as tenure, contract type, internet service, billing details, and payment method. The project includes data cleaning, missing value handling, skew correction, categorical encoding, class imbalance treatment using SMOTE, and building a **Random Forest Classifier** for churn prediction. Model performance was evaluated using key classification metrics such as **precision, recall, and confusion matrix** to understand how effectively the model identifies potential churners.
